# Simple Interest Calculator with CI/CD Setup

A responsive, user-friendly web application to calculate Simple Interest using the formula: SI = (P * R * N) / 100.

## Project Details
- **Frontend Framework:** Tailwind CSS
- **Logic:** Vanilla JavaScript
- **Deployment Platform:** GitHub & Jenkins Execution Environment

## Local Setup Instructions
1. Clone this repository to your local system.
2. Open the `index.html` file directly in any modern web browser (Chrome, Edge, Firefox).
3. Input values for Principal (P), Rate (R), and Time (N) to see instantaneous calculations.
